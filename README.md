Unser Schulspiel Multiboard. 
Es geht darum, dass Lehrer auf dem Server H5P bzw. normale Aufgaben in H5P Räume hochladen kann, die auf Boards gespielt werden können. So kann der Lehrer H5PP Dateien nicht immer raussuchen sondern hat sie zentral auf einem Fleck. 
Funktionsweise mithilfe von fastApi, websockets sowie H5P Standalone.
Setup
Github clonen, cd server für server bzw cd client für client-. In beiden Fällen pip install -r requirement.txt Danach erst Server starten mit gewünschter Ip über Ipconfig vom Server. Im client IP vom Server eingeben. 
Starten kann es dann mit main.py
